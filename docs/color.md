# color

## Properties

#### title:

The title of color control

- Type: `string`
- Required: No
- Default: 'Color'

#### initialOpen:

- Type: `bool`
- Required: No

For more read gutenberg [readme](https://github.com/WordPress/gutenberg/tree/master/packages/editor/src/components/color-palette).

**Example:**

```js
color: {
	type: 'string',
	field: {
		type: 'color',
		placement: 'inspector',
	},
}
```

### Return

This will return color hex code. For eg. `#000000`

- Type: `string`
